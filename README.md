# EtiKett

EtiKett is an open-source repository that documents regional eating habits, meal etiquettes, and dining traditions across India, starting with Kerala. It aims to preserve and share how people eat, serve, and celebrate food — from the layout of a sadya to the customs around hand-eating, sharing, and serving others.

This project welcomes contributions from anyone interested in food anthropology, regional culture, or everyday social practices. Together, we can build a living record of India’s culinary manners, meal rituals, and the stories that surround them.
